# Notebook Guidance

The original rough notebook is not published because it contained draft-specific material that is not suitable for a public portfolio repository.

For a portfolio-safe notebook, use the scripts in `src/` as the source of truth and create a concise walkthrough with:

1. Dataset structure check.
2. Example preprocessing/augmentation.
3. Model architecture summary.
4. Training command and logged metrics.
5. Evaluation metrics and confusion matrix.
6. Limitations and next steps.

Keep the notebook independent, reproducible and free of private assessment instructions.
